# Movie Reviews Project

A Django-based web application for users to browse and review movies.

## Features

*   Browse a collection of movies.
*   User registration and authentication.
*   Write and post reviews for movies.
*   View reviews from other users.

## Technologies Used

*   Python
*   Django
*   SQLite

## Setup and Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/moviereviewsproject.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd moviereviewsproject
    ```
3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
4.  **Apply migrations:**
    ```bash
    python manage.py migrate
    ```
5.  **Run the development server:**
    ```bash
    python manage.py runserver
    ```

## Usage

Once the server is running, you can access the application in your web browser at `http://127.0.0.1:8000/`.
